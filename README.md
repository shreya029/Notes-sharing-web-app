# NoteShare — Notes Sharing Web App

A web application where users can create, edit, delete, and share notes with others via a link.

## Features

- User login and registration
- Create, edit, and delete notes
- Share notes with anyone via a public link
- Search through your notes
- Trash and restore deleted notes

## Tech Stack

- React.js
- Firebase (Auth + Firestore)
- CSS Modules
